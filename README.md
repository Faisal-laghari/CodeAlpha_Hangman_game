# CodeAlpha Hangman Game

Description

This is a simple text-based Hangman game developed in Python as part of the CodeAlpha Python Programming Internship.

# Features

- Random word selection
- User guesses one letter at a time
- Maximum of 6 incorrect attempts
- Displays remaining chances
- Beginner-friendly and easy to understand

# Technologies Used

- Python
- random module

Concepts Used

- Functions
- Loops
- Conditional Statements
- Strings
- Lists
- Input and Output

# How to Run

1. Download the project.
2. Open the folder in VS Code.
3. Run the following command:

python Hangman_game.py
